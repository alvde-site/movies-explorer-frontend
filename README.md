# movies-explorer-frontend
Дипломная работа Демиденко Александра на Яндекс Практикуме.
Технологии используемые в проекте:
* Flexbox, Grid
* Верстка по БЭМ(Nested)
* Адаптивная верстка
* Figma
* JS
* popup
* Валидация формы
* ООП
* JS Классы
* webpack
* React, useState, useEffect, useHistory
